# ⭕ 円の面積計算プログラム (Circle Area Calculator)

C言語を用いて、入力された半径から円の面積を算出するプログラムです。

`#C言語` `#数学` `#幾何学` `#自作関数`

## 📝 概要
このプログラムは、円の半径を入力することで、高精度な円周率（π）を用いて面積を計算します。二乗計算を独立した関数として定義しており、数式の見通しが良い構造になっています。

## ✨ 特徴
* **高精度定数の定義**: `#define` を使用し、円周率を小数点以下31位まで定義することで計算の正確性を確保しています。
* **関数のモジュール化**: 二乗計算を `sqr` 関数として独立させており、メインロジック（面積計算）の可読性を高めています。
* **シンプルなインターフェース**: 標準入力から半径を受け取り、即座に結果を返す直感的な操作感です。

## 🚀 操作内容
1. 実行後、`radius = ` と表示されたら半径の数値を入力します。
2. 計算結果が `S = [面積]` の形式で表示されます。

## 📂 ファイル構成
* `circle_area.c`: 面積計算ロジックと二乗関数を含むプログラム。

## 💻 実行方法
```bash
gcc -o circle_area circle_area.c
./circle_area
