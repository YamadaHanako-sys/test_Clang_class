# 📊 成績データ処理・ソートプログラム

ファイルから成績データを読み込み、条件に合うデータのみを抽出してカナ順にソートするプログラムです。

`#C言語` `#ファイル操作` `#構造体` `#選択ソート` `#データ処理`

## 📝 概要
このプログラムは、外部ファイル（`score.d`）に保存された氏名、カナ、5教科の点数データを読み込みます。
特定の合計点数を超えた学生のみを抽出し、さらにそのリストを**カタカナの辞書順（五十音順）**に並び替えて、画面および新しいファイルに出力します。

## ✨ 特徴
* **ファイル入出力**: `fopen` や `fscanf` / `fprintf` を用いた、実践的なファイル操作を実装しています。
* **データフィルタリング**: 合計点が300点を超えるデータのみを保持するロジックを組み込んでいます。
* **選択ソート（Selection Sort）**: `strcmp` 関数を利用して、文字列（カナ）を比較し、正確に並び替えます。
* **構造体による管理**: 氏名から各科目、合計点までを一括で管理する `struct kamoku` を定義しています。

## 🚀 操作内容
1. **入力ファイル名の指定**: 実行後、読み込むファイル名（例：`score.d`）を入力します。
2. **出力ファイル名の指定**: 保存先のファイル名（例：`sorted.d`）を入力します。
3. **自動処理**: プログラムがデータを読み込み、300点以下の除外とソートを自動で行います。
4. **結果表示**: 画面に整形された表が表示され、同時にファイルへ保存されます。

## 📂 ファイル構成
* `sort.c`: 成績処理・ソートの本体プログラム。
* `score.d`: 入力用データファイル（氏名、カナ、点数データ）。

## 💻 実行方法
```bash
# コンパイル
gcc -o sort sort.c

# 実行
./sort

# 実行時の入力例
input file name：score.d
output file name：sorted.d
