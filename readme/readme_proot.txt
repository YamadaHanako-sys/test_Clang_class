# 📈 内部収益率 (IRR) 計算プログラム

C言語を用いて、二分法（Bisection Method）により多項式の根を求め、内部収益率を算出するプログラムです。

`#C言語` `#数値計算` `#二分法` `#金融工学` `#IRR`

## 📝 概要
このプログラムは、投資プロジェクトの収益性を評価する指標である**内部収益率 (Internal Rate of Return: IRR)** を計算します。多項式の根を探索するアルゴリズムとして**二分法**を採用しており、指定された誤差範囲内で近似解を求めます。

## ✨ 特徴
* **二分法アルゴリズム**: 探索範囲を半分ずつ絞り込んでいく堅牢なアルゴリズムを実装しています。
* **ホーナー法の応用**: 多項式の値を効率的に計算するために、計算負荷の少ない累次的な計算手法を採用しています。
* **柔軟な次数対応**: 最大サイズ（`SIZE`）の範囲内で、任意の次数の多項式に対して計算が可能です。



## 🚀 操作内容
* **係数の入力**: 多項式の各係数（`a[0]`〜）を順番に入力します。
* **範囲指定**: `LOW`（1.0）から `HIGH`（3.0）の範囲内で根を探索します。
* **精度の制御**: `ERR` 定数により、`0.000001` 単位の高精度な計算を行います。

## 📂 ファイル構成
* `irr_calc.c`: 二分法および多項式計算のロジックを含むメインプログラム。
* （必要に応じて `irr_calc.h` や `main.c` に分割して管理します）
