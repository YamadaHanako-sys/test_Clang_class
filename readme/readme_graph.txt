# 📉 二次関数可視化プログラム

C言語を用いて、二次関数の計算結果をアスキーアート形式でグラフ化するプログラムです。

`#C言語` `#数学` `#二次関数` `#可視化` `#アルゴリズム`

## 📝 概要
このプログラムは、二次関数 $y = x^2 + 4$ の計算を行い、その結果をターミナル上に視覚的に表示します。
計算された値（y）の数だけ文字 `a` を出力することで、関数の形状（放物線）を横向きのグラフとして確認できます。

## ✨ 特徴
* **数式の実装**: 関数 `nizi` により、特定の数式ロジックを独立させて管理しています。
* **視覚的出力**: 数値を表示するだけでなく、ループ処理を重ねることでグラフのような視覚化を行っています。
* **制御文の活用**: `for` 文による範囲指定と、ネスト（入れ子）構造による描画処理を組み合わせています。

## 🚀 操作内容
* **計算範囲**: $x = -7$ から $x = 7$ までの範囲を自動で計算します。
* **グラフ表示**: 
  - 左端に $x$ の値を表示。
  - 右側に $y$ の値に比例した数の `a` を表示。

## 📂 ファイル構成
* `quadratic_graph.c`: 関数計算と描画ロジックを含むメインプログラム。

## 💻 実行方法
```bash
# コンパイル
gcc -o quadratic_graph quadratic_graph.c

# 実行
./quadratic_graph
