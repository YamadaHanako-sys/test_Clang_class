# 📐 三角形面積計算プログラム

C言語で、3辺の長さからヘロンの公式を用いて面積を求めるプログラムです。

`#C言語` `#数学` `#ヘロンの公式` `#三角形`

## 📝 概要
3つの辺の長さ（a, b, c）を入力し、その数値が三角形として成立するかを判定した上で、面積を算出します。

## ✨ 特徴
* **成立条件の判定**: `if` 文を用いて、入力された数値が数学的に三角形を作れるかチェックしています。
* **数学関数の利用**: `math.h` を読み込み、平方根（`sqrt`）や絶対値（`fabs`）を使用して計算しています。

## 🚀 操作内容
1. 実行後、3つの数値をスペース区切りで入力します。
2. 三角形が成立すれば面積が表示され、成立しなければ `No triangle.` と表示されます。

## 📂 ファイル構成
* `triangle_area.c`: プログラム本体。

## 💻 実行方法
```bash
gcc -o triangle_area triangle_area.c -lm
./triangle_area
