#二分木探索プログラム（C言語）

C言語による二分探索木のスクラッチ実装です。

## 概要
このプロジェクトは、基本的なデータ構造である二分探索木をC言語で実装したものです。
独自のデータ構造定義から、ノードの追加、削除、再帰的な木構造の表示までの一連の機能を備えています。

## 制作背景
ライブラリに頼らず、データ構造の内部動作を深く理解するために制作しました。
特に以下の技術的要素に重点を置いています。
* **ポインタの深い理解**: 構造体ポインタやダブルポインタを用いたノードの繋ぎ替え。
* **メモリ管理**: `calloc` による動的確保と `free` による適切なメモリ解放。
* **アルゴリズムの実装**: 再帰処理を用いた木の走査や、削除時のノードの再配置ロジック。

## 機能
* **データの追加 (`add`)**: 二分探索木のルールに基づき、適切な位置にノードを挿入します。
* **データの削除 (`remove_node`)**: 
  * 子を持たないノード
  * 子を1つ持つノード
  * 子を2つ持つノード（左部分木の最大値を探索して移動）
  それぞれのケースに対応した削除ロジックを実装しています。
* **木構造の表示 (`printtree`)**: 親子関係が視覚的にわかる形式でターミナルに出力します。

## ファイル構成
| ファイル名 | 役割 |
| :--- | :--- |
| `main.c` | プログラムの実行例（エントリーポイント） |
| `bintree.c` / `.h` | 二分探索木の基本ロジックと構造体定義 |
| `mydata.c` / `.h` | 木に格納する具体的なデータ（ID・名前）の操作 |

## 実行方法

コンパイルには `gcc` などのCコンパイラを使用してください。

```bash
# コンパイル
gcc -o bst_demo main.c bintree.c mydata.c

# 実行
./bst_demo
