# 🔍 Linear Search プログラム

[cite_start]C言語による線形探索（逐次探索）の実装です [cite: 1]。

`#C言語` `#アルゴリズム` `#学習用` `#線形探索` `#メモリ管理`

## 📝 概要
[cite_start]線形探索（Linear Search）は、リストの先頭から順番に目的のデータを探していく最も基本的な探索アルゴリズムです [cite: 2]。データがソート（整列）されていない状態でも、確実に目的の要素を見つけることができます。

## ✨ 特徴
* **シンプルなロジック**: 配列の全要素を $O(n)$ の計算量で走査する、理解しやすく堅牢なアルゴリズムです。
* **動的メモリ管理**: `calloc` を使用して実行時に必要な分だけメモリを確保し、終了時に `free` で確実に解放する安全な設計です。
* **汎用的な設計**: 探索ロジックを独立した関数（`search`）として切り出しています。

## 🚀 操作内容
* **要素の入力**: ユーザーが指定した個数分のデータを配列に格納します。
* **ターゲット指定**: 探索したい値（キー）を入力します。
* **探索と結果表示**: 配列内を先頭からスキャンし、見つかった場合はそのインデックスを表示し、見つからない場合はエラーメッセージを表示します。

## 📂 ファイル構成
* `linear_search.c`: 線形探索の本体ロジック（`search` 関数）。
* `linear_search.h`: 関数のプロトタイプ宣言。
* [cite_start]`main.c`: ユーザー入力を受け取り、探索を実行するデモンストレーション [cite: 3]。

## 💻 実行方法
コンパイルには `gcc` などのCコンパイラを使用してください。

```bash
# コンパイル
gcc -o linear_search main.c linear_search.c

# 実行
./linear_search
