# 📊 統計処理プログラム (stats.c)

C言語を用いた、標準入力からのデータ集計および簡易統計処理の実装です。

`#C言語` `#統計` `#データ処理` `#基礎アルゴリズム`

## 📝 概要
入力された一連の数値データ（点数など）を読み込み、データの総数、および特定のしきい値に基づいたカテゴリ別の集計を行う統計プログラムです。

## ✨ 特徴
* **リアルタイム集計**: `while` ループと `EOF` 判定を組み合わせ、データの個数にかかわらず柔軟に統計処理が可能です。
* **条件ベースの統計（Stats）**: 80点以上（High score）や60点未満（Low score）といった特定の条件に合致するデータ数を抽出します。
* **クリーンな実装**: 最小限の変数と論理構造で、読みやすさと実行速度を両立させています。

## 🚀 操作内容
1. 実行後、数値を一つずつ入力し、Enterを押します。
2. 全てのデータを入力し終えたら、`Ctrl + Z`（Windows）または `Ctrl + D`（Mac/Linux）を入力して統計を確定させます。
3. 最終的なデータの総数と、各カテゴリの集計結果が表示されます。

## 📂 ファイル構成
* `stats.c`: 統計集計ロジックを含むメインプログラム。

## 💻 実行方法
```bash
gcc -o stats stats.c
./stats
