# 💰 元金算出プログラム (Principal Calculator)

C言語を用いて、目標とする満期金額から必要な元金を複利計算で算出するプログラムです。

`#C言語` `#金融工学` `#複利計算` `#数理モデル`

## 📝 概要
このプログラムは、将来の目標金額（満期金額）、年利、および運用期間を入力することで、現在いくらの元金が必要かを算出します。複利（Compound Interest）の概念に基づいた計算を行っています。

## ✨ 特徴
* **複利ロジックの実装**: `for` ループを用いたべき乗計算により、複利の効果をシミュレーションしています。
* **実務的な変数設計**: 利率、期間、金額といった実務で使われるパラメータを構造的に処理しています。
* **逆算アルゴリズム**: 通常の将来価値計算ではなく、現在価値（元金）を求める逆算プロセスを実装しています。

## 🚀 操作内容
1. 実行後、`Maturity_amount`（目標額）、`interest_rate`（年利 %）、`years`（期間）をスペース区切りで入力します。
2. 入力に基づき、必要な元金（Principal）が整数値で算出され、入力データと共に表示されます。

## 📂 ファイル構成
* `principal_calc.c`: 複利の逆算ロジックを含むメインプログラム。

## 💻 実行方法
```bash
gcc -o principal_calc principal_calc.c
./principal_calc
