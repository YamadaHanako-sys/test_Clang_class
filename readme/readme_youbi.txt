# 📅 日数計算プログラム (Day Counter)

C言語を用いて、2つの日付の間の日数を正確に計算するプログラムです。

`#C言語` `#アルゴリズム` `#数値計算` `#カレンダー` `#ツェラーの公式`

## 📝 概要
このプログラムは、入力された2つの日付（年・月・日）の間の日数を算出します。グレゴリオ暦に基づき、閏年（4年ごと、100年ごと、400年ごとのルール）を考慮した精密な計算を行います。

## ✨ 特徴
* **高精度な計算**: 西暦0年を基準とした通算日数を算出するアルゴリズムを採用しています。
* **閏年への完全対応**: 1月と2月を前年の13月・14月として扱うことで、複雑な閏年の判定を数式内で簡潔に処理しています。
* **繰り返し計算**: プログラムを終了させずに、複数の日付ペアを連続して計算可能です。

## 🚀 操作内容
1. **日付の入力**: `Y1 M1 D1, Y2 M2 D2` の形式で、スペース区切りで2つの日付を入力します。
   - 例: `2024 1 1 2024 12 31`
2. **結果の確認**: 入力した2つの日付の差が日数として表示されます。
3. **終了方法**: Windowsでは `Ctrl + Z`、Mac/Linuxでは `Ctrl + D` を入力すると終了します。

## 📂 ファイル構成
* `day_counter.c`: 日数計算ロジックとメイン処理を含むプログラム。

## 💻 実行方法
```bash
# コンパイル
gcc -o day_counter day_counter.c

# 実行
