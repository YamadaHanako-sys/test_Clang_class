#stackプログラム

C言語によるスタックデータ構造の実装です。

## 概要
スタック（Stack）は、**LIFO (Last-In, First-Out: 後入れ先出し)** の原則に基づくデータ構造です。本実装では、動的に確保した配列を用いて、整数の格納と取り出しを行います。

## 特徴
* **スタックポインタ管理**: 構造体メンバ `ptr` を用いて、現在のデータ位置とスタックの空き状況を効率的に管理しています。
* **動的メモリ確保**: `calloc` を使用し、実行時に指定されたサイズでスタックを生成します。
* **エラーハンドリング**: スタックオーバーフロー（満杯）やスタックアンダーフロー（空）の状態を適切に判定し、エラー値を返す設計にしています。

## 操作内容
* `push`: データをスタックの最上部に追加します。
* `pop`: スタックの最上部からデータを取り出します。
* `print`: 現在スタックに積まれている内容を一覧表示します。



## ファイル構成
* `stack.c`: スタック操作の本体ロジック。
* `stack.h`: スタック構造体および関数の定義。
* `main.c`: スタック操作のデモンストレーション。
