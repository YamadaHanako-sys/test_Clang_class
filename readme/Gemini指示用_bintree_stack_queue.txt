## プロジェクトの概要
プログラミングの基礎であるデータ構造とアルゴリズムを、標準ライブラリに頼らずC言語で一から実装したプロジェクトです。ポインタ操作や動的なメモリ管理など、低レイヤーの処理を意識したコードを記述しています。

## 実装したデータ構造

### 1. Stack（スタック）
* [cite_start]**方式**: LIFO (Last-In, First-Out: 後入れ先出し) [cite: 5]
* **特徴**: 
  * [cite_start]`calloc` による動的メモリ確保 [cite: 5]
  * [cite_start]スタックポインタ `ptr` による厳密な要素管理 [cite: 5]
* [cite_start]**操作内容**: `push`（挿入）、`pop`（取り出し）、`print`（表示） [cite: 5]

### 2. Queue（キュー）
* [cite_start]**方式**: FIFO (First-In, First-Out: 先入れ先出し) [cite: 1]
* **特徴**: 
  * [cite_start]**リングバッファ（循環配列）** を採用し、データ移動のオーバーヘッドを抑制 [cite: 1]
  * [cite_start]`front` と `rear` の2つのインデックスを用いたポインタ制御 [cite: 1]
* [cite_start]**操作内容**: `Enque`（挿入）、`Deque`（取り出し）、`Print`（表示） [cite: 1]

### 3. Binary Search Tree（二分探索木）
* [cite_start]**概要**: 独自のデータ構造定義から、再帰的な木構造の表示までを網羅 [cite: 2]
* **特徴**: 
  * [cite_start]子ノードの数に応じた複雑なノード削除アルゴリズムの実装 [cite: 2]
  * [cite_start]ポインタの繋ぎ替えによる動的な木構造の維持 [cite: 2]
* [cite_start]**操作内容**: `add`（挿入）、`remove_node`（削除）、`printtree`（階層表示） [cite: 2]
