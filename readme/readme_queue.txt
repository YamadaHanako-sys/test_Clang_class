# Queueプログラム

C言語によるキューデータ構造の実装です。

## 概要
キュー（Queue）は、**FIFO (First-In, First-Out: 先入れ先出し)** の原則に基づくデータ構造です。本実装では、効率的なメモリ利用を目的として**リングバッファ（循環配列）**を採用しています。

## 特徴
* **リングバッファ（循環配列）**: 配列を円状に使用することで、データの取り出し（デキュー）時に他の要素をずらす必要がなく、処理速度 $O(1)$ を実現しています。
* **ポインタ制御**: `front`（先頭）と `rear`（末尾）の2つのインデックスを管理し、配列の終端に達した際に先頭へ戻るロジックを実装しています。
* **状態管理**: 現在の要素数 `num` を保持することで、満杯か空かを即座に判断可能です。

## 操作内容
* `Enque`: データをキューの末尾に追加します。
* `Deque`: キューの先頭からデータを取り出します。
* `Print`: リングバッファの論理的な順序に従って要素を表示します。



## ファイル構成
* `queue.c`: キュー操作（リングバッファ）の本体ロジック。
* `queue.h`: キュー構造体および関数の定義。
* `main.c`: キュー操作のデモンストレーション。
